== Description ==

The official HBAgency (https://www.hbagency.it) plugin makes the integration process effortless.
 
With this plugin, you can:
 - Easily integrate your WordPress website with our advanced technology.
 - Automatically insert and update ads.txt, scripts, and ad placements.
 - Manage placements and edit additional lines in your ads.txt file with ease.
